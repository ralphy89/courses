Exercices JavaScript — Appareils électroniques

Contexte : appareils électroniques. Ex. 1 : variables et formulaire. Ex. 2 : plusieurs appareils dans un tableau, chacun décrit par un objet.

Exercice 1 — Variables et formulaire

Lire les champs d’un formulaire et les stocker dans des variables.

Consignes (formateur / élève)

  1. Dans le fichier index.js, repérez le gestionnaire submit du formulaire principal.
  2. Pour chaque champ, créez une variable (const ou let) qui lit .value sur l’élément correspondant (ex. document.getElementById("device-name")).
  3. Pour le prix, convertissez le texte en nombre avec Number(...) ou parseFloat(...).
  4. Pour la case « Appareil neuf », utilisez .checked (booléen true / false).
  5. Construisez une chaîne (template literal) et affichez-la dans #output avec textContent.
  6. Option : affichez aussi les variables dans la console avec console.log.
  7. Bonus (déjà câblé dans le script) : bouton Mode clair — une variable booléenne et document.documentElement.setAttribute("data-theme", "light") pour changer les couleurs de la page.

Formulaire

Exemple de cœur du script 
form.addEventListener("submit", function (event) {
  event.preventDefault();

  const nomAppareil = document.getElementById("device-name").value.trim();
  const marque = document.getElementById("brand").value.trim();
  const prix = Number(document.getElementById("price").value);
  const type = document.getElementById("category").value;
  const estNeuf = document.getElementById("is-new").checked;

  const resume =
    `Appareil : ${nomAppareil}\n` +
    `Marque : ${marque}\n` +
    `Prix : ${prix} €\n` +
    `Type : ${type}\n` +
    `Neuf : ${estNeuf ? "oui" : "non"}`;

  document.getElementById("output").textContent = resume;
  console.log({ nomAppareil, marque, prix, type, estNeuf });
});

Exercice 2 — Liste (tableau) et objets

Regrouper plusieurs appareils dans un tableau ; chaque appareil est un objet { nom, marque, prix, type }. Ajouter au stock puis réafficher la liste.

Consignes (formateur / élève)

  1. Dans index.js, section Exercice 2, créez un tableau vide let inventaire = [] (ou avec quelques objets de démo).
  2. Sur envoi du formulaire « Ajouter au stock », construisez un objet littéral avec les propriétés nom, marque, prix (nombre), type (chaîne).
  3. Ajoutez cet objet au tableau avec .push(...).
  4. Écrivez une fonction afficherInventaire() qui vide la liste HTML puis parcourt le tableau (boucle for) pour créer un <li> par appareil (texte lisible ou textContent).
  5. Rappelez afficherInventaire() après chaque ajout.
  6. Option : affichez inventaire.length quelque part (nombre d’articles en stock).

Stock (tableau d’objets)

    Exemple — tableau, objet, push, boucle 
    let inventaire = [];

function afficherInventaire() {
  const liste = document.getElementById("stock-list");
  const compteur = document.getElementById("stock-count");
  liste.innerHTML = "";
  compteur.textContent = inventaire.length + " appareil(s) en stock";

  for (let i = 0; i < inventaire.length; i++) {
    const a = inventaire[i];
    const li = document.createElement("li");
    li.textContent =
      a.nom + " — " + a.marque + " — " + a.prix + " € — " + a.type;
    liste.appendChild(li);
  }
}

document.getElementById("stock-form").addEventListener("submit", function (event) {
  event.preventDefault();
  const appareil = {
    nom: document.getElementById("stock-device-name").value.trim(),
    marque: document.getElementById("stock-brand").value.trim(),
    prix: Number(document.getElementById("stock-price").value),
    type: document.getElementById("stock-category").value,
  };
  inventaire.push(appareil);
  document.getElementById("stock-form").reset();
  afficherInventaire();
});
