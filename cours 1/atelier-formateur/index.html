Exercices JavaScript — Appareils électroniques

Contexte : appareils électroniques. Ex. 1 : variables et formulaire. Ex. 2 : plusieurs appareils dans un tableau ; ex. 2 : objets avec image, fiches en cartes.

Exercice 1 — Variables et formulaire

Lire les champs d’un formulaire et les stocker dans des variables.

Consignes (formateur / élève)

  1. Dans le fichier index.js, repérez le gestionnaire submit du formulaire principal.
  2. Pour chaque champ, créez une variable (const ou let) qui lit .value sur l’élément correspondant (ex. document.getElementById("device-name")).
  3. Pour le prix, convertissez le texte en nombre avec Number(...) ou parseFloat(...).
  4. Pour la case « Appareil neuf », utilisez .checked (booléen true / false).
  5. Construisez une chaîne (template literal) et affichez-la dans #output avec textContent.
  6. Option : affichez aussi les variables dans la console avec console.log.
  7. Bonus (déjà câblé dans le script) : bouton Mode clair — une variable booléenne et document.documentElement.setAttribute("data-theme", "light") pour changer les couleurs de la page.

Formulaire

Exemple de cœur du script 
form.addEventListener("submit", function (event) {
  event.preventDefault();

  const nomAppareil = document.getElementById("device-name").value.trim();
  const marque = document.getElementById("brand").value.trim();
  const prix = Number(document.getElementById("price").value);
  const type = document.getElementById("category").value;
  const estNeuf = document.getElementById("is-new").checked;

  const resume =
    `Appareil : ${nomAppareil}\n` +
    `Marque : ${marque}\n` +
    `Prix : ${prix} €\n` +
    `Type : ${type}\n` +
    `Neuf : ${estNeuf ? "oui" : "non"}`;

  document.getElementById("output").textContent = resume;
  console.log({ nomAppareil, marque, prix, type, estNeuf });
});

Exercice 2 — Inventaire en cartes (image + infos)

Tableau d’objets { imageUrl, modele, marque, categorie, prix, quantite } — types variés (téléphone, laptop, console…). Formulaire à gauche, cartes à droite (comme templates-eleves/exo2.html).

Consignes (formateur / élève)

  1. Dans index.js, section Exercice 2 : let inventaire = [].
  2. À la soumission : lire les champs #phone-image (URL facultative), #phone-modele, #phone-marque, #phone-categorie, #phone-prix, #phone-quantite ; valider avec if (messages dans #stock-erreur).
  3. inventaire.push({ ... }), reset(), puis afficherCartes().
  4. afficherCartes() : vider #stock-cartes, boucle for, chaque article = <article class="stock-card"> avec .stock-card-media (<img> ou .stock-card-photo-placeholder) et .stock-card-body (modèle, marque, type, prix, stock).
  5. Afficher inventaire.length dans #stock-count.

Ajouter un appareil

Exemple — push, cartes, image ou placeholder 
function afficherCartes() {
  const zone = document.getElementById("stock-cartes");
  const compteur = document.getElementById("stock-count");
  zone.innerHTML = "";
  compteur.textContent = inventaire.length + " référence(s) en inventaire";

  for (let i = 0; i < inventaire.length; i++) {
    const a = inventaire[i];
    const card = document.createElement("article");
    card.className = "stock-card";

    const media = document.createElement("div");
    media.className = "stock-card-media";
    if (a.imageUrl && a.imageUrl.trim() !== "") {
      const img = document.createElement("img");
      img.src = a.imageUrl.trim();
      img.alt = "Photo de " + a.modele;
      media.appendChild(img);
    } else {
      const ph = document.createElement("div");
      ph.className = "stock-card-photo-placeholder";
      ph.textContent = "📱";
      media.appendChild(ph);
    }
    card.appendChild(media);

    const body = document.createElement("div");
    body.className = "stock-card-body";
    const titre = document.createElement("h3");
    titre.textContent = a.modele;
    body.appendChild(titre);
    /* … paragraphes marque, catégorie, prix, quantité avec .textContent … */
    zone.appendChild(card);
  }
}
