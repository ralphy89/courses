1. Introduction à JavaScript

JavaScript est le langage qui permet d’ajouter de la logique et de l’interactivité à une page web. Avec HTML, on structure le contenu. Avec CSS, on améliore l’apparence. Avec JavaScript, on rend la page vivante.

Technologie Rôle Image simple
HTML Structure de la page Le squelette
CSS Style et design Les vêtements
JavaScript Comportement et interaction Le mouvement

Où exécuter le code ?

Dans le navigateur, via la console et les fichiers .js liés à votre page HTML.

Premier test

console.log("Bonjour JavaScript !");
Utilisez F12 puis l’onglet Console pour voir vos messages et tester rapidement votre code.

2. Projet fil rouge — Créer un portfolio dynamique

Au lieu d’apprendre les notions de manière isolée, nous allons construire un petit portfolio interactif. Chaque concept sera appliqué dans ce projet.

Fonctionnalités du projet

  • Afficher un nom et une bio
  • Afficher une liste de compétences
  • Regrouper les informations dans un objet
  • Utiliser JavaScript pour remplir la page
Démo dans ce dépôt — ouvrez portfolio-demo/index.html (fichiers index.html, style.css, script.js), même logique que ci-dessous.

Structure HTML de base

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8" />
  <title>Mon Portfolio</title>
</head>
<body>
  <h1 id="nom"></h1>
  <p id="bio"></p>

  <h3>Mes compétences :</h3>

  <script src="script.js"></script>

</html>
Dans ce projet, les données du portfolio seront injectées dans la page avec JavaScript.

3. Les variables

Une variable permet de stocker une information. En JavaScript, on utilise surtout let et const.

Mot-clé Utilisation Réassignable ?
let Variable qui peut changer Oui
const Valeur fixe ou référence stable Non

Exemple simple

let nom = "Ralph";
let bio = "Développeur web débutant";

console.log(nom);
console.log(bio);

Application dans le projet

Ces variables vont contenir les premières informations du portfolio. 

let nom = "Ralph";
let bio = "Développeur web débutant";
Une variable ne sert pas seulement à stocker une valeur. Elle permet de réutiliser cette valeur plus tard dans le projet.

4. Les tableaux

Un tableau permet de stocker plusieurs valeurs dans une seule variable. C’est idéal pour représenter une liste, comme des compétences dans un portfolio.

Exemple

let skills = ["HTML", "CSS", "JavaScript"];

console.log(skills);
console.log(skills[0]); // HTML

Pourquoi un tableau ?

Parce qu’un portfolio contient souvent plusieurs compétences, plusieurs projets ou plusieurs liens.

Idée clé

Un tableau regroupe des éléments similaires que l’on peut parcourir plus facilement.

Application dans le projet

let skills = ["HTML", "CSS", "JavaScript"];

Chaque élément du tableau représente une compétence que l’on affichera dans la page.

5. Les objets

Un objet permet de regrouper plusieurs informations liées dans une seule structure. C’est parfait pour modéliser un profil.

Exemple

let profil = {
  nom: "Ralph",
  bio: "Développeur web débutant",
  ville: "Gonaïves"
};

console.log(profil.nom);
console.log(profil.ville);

Application dans le projet

Au lieu de créer plusieurs variables séparées, on regroupe toutes les informations principales dans un seul objet. 

let profil = {
  nom: "Ralph",
  bio: "Développeur web débutant",
  ville: "Gonaïves"
};
Un objet est utile quand plusieurs informations appartiennent à la même entité. Ici, l’entité est le profil.

6. Manipuler le DOM

Le DOM représente la page HTML sous forme d’objets manipulables. Avec JavaScript, on peut sélectionner des éléments, modifier leur texte, créer de nouveaux éléments et les ajouter dans la page.

Fonction principale du projet

function afficherPortfolio() {
  document.getElementById("nom").textContent = profil.nom;
  document.getElementById("bio").textContent = profil.bio;

  let liste = document.getElementById("skills");
  liste.innerHTML = "";

  for (let i = 0; i < skills.length; i++) {
    let li = document.createElement("li");
    li.textContent = skills[i];
    liste.appendChild(li);
  }
}

Explication

  • getElementById() sélectionne un élément HTML
  • textContent modifie le texte affiché
  • createElement() crée un nouvel élément HTML
  • appendChild() ajoute cet élément dans la page
C’est ici que JavaScript devient visible pour l’utilisateur. Les données quittent le code et apparaissent dans l’interface.

7. Code final du mini portfolio

Voici une version complète du script JavaScript pour le projet. 

let profil = {
  nom: "Ralph",
  bio: "Développeur web débutant",
  ville: "Gonaïves"
};

let skills = ["HTML", "CSS", "JavaScript"];

function afficherPortfolio() {
  document.getElementById("nom").textContent = profil.nom;
  document.getElementById("bio").textContent = profil.bio + " • " + profil.ville;

  let liste = document.getElementById("skills");
  liste.innerHTML = "";

  for (let i = 0; i < skills.length; i++) {
    let li = document.createElement("li");
    li.textContent = skills[i];
    liste.appendChild(li);
  }
}

Étapes du fonctionnement

  1. L’utilisateur clique sur le bouton.
  2. La fonction afficherPortfolio() est exécutée.
  3. Le nom et la bio sont injectés dans la page.
  4. Les compétences sont parcourues puis ajoutées à la liste.
