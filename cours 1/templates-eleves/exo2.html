Exercice 2 — Stock d’appareils électroniques (cartes)

Smartphones, laptops, tablettes, consoles, audio… Chaque article est un objet avec image, modèle, marque, type, prix et quantité. Le formulaire est à gauche ; les cartes s’affichent à droite.

Script à compléter : exo2.js.

Ressource — Pour retrouver marque, modèle, prix indicatif ou photos d’appareils (téléphones, laptops, consoles, audio…), vous pouvez vous appuyer sur les fiches du site Les Numériques (tests et comparatifs, en français). En exercice, les valeurs peuvent être approximatives ; pour une image, un lien direct vers une photo du produit ou un service d’images libres convient.

Consignes

  1. let inventaire = [] dans exo2.js.
  2. Objet à enregistrer (exemple) : { imageUrl, modele, marque, categorie, prix, quantite } — categorie reprend le type choisi (téléphone, laptop, console…). imageUrl peut être vide.
  3. À la soumission : if pour modèle, marque et type non vides ; prix et quantité valides (nombres ≥ 0). Sinon message dans #stock-erreur.
  4. Sinon : push, vider l’erreur, reset() du formulaire, afficherCartes().
  5. afficherCartes() : vider #stock-cartes, boucle for, pour chaque article un <article class="stock-card"> avec une image (<img> si URL non vide, sinon la zone .stock-card-photo-placeholder) et toutes les infos (marque, type, prix, stock).
  6. Mettre à jour #stock-count avec inventaire.length.
