Exercice 1 — Variables et formulaire

Contexte : appareil électronique. Lire le formulaire et afficher un récapitulatif avec des variables JavaScript.

Script à compléter : exo1.js (ce fichier HTML charge ce script en bas de page).

Consignes

  1. Dans exo1.js, repérez le gestionnaire submit du formulaire.
  2. Pour chaque champ, une variable (const ou let) qui lit .value (ex. document.getElementById("device-name")).
  3. Prix : Number(...) ou parseFloat(...).
  4. Case « Appareil neuf » : .checked (booléen).
  5. Récapitulatif dans #output avec textContent (template literal conseillé).
  6. Le thème clair / sombre est déjà géré dans exo1.js (bonus).

Formulaire

Ce dossier : exo1.html, exo1.js, style.css (commun) + exo2.
