Projet — To‑Do List

Construisez une application web simple pour gérer une liste de tâches : ajouter, afficher, supprimer. Bonus : terminer, vider la liste, sauvegarder avec localStorage.

Thèmes

DOM createElement appendChild addEventListener tableaux fonctions localStorage (bonus)
Durée indicative : 2h (1h30 dev + 30 min présentation).

Objectif

Ajouter Saisir une tâche et l’ajouter à la liste.
Afficher Voir toutes les tâches dans la page.
Supprimer Retirer une tâche de la liste.

Sommaire

Introduction Spécifications Plan de réalisation Bonus Présentation & évaluation Aide (rappels DOM) Récap & suite Template élève

Introduction

Dans ce projet, vous allez gérer une liste de tâches dans un tableau JavaScript, puis afficher ce tableau dans le HTML en créant des éléments avec le DOM.

Template de départ : ouvrez cours 4/templates-eleves/ puis lancez todo.html. Ouvrir le template élève.

Spécifications (obligatoires)

Fonctionnalités

  • Ajouter une tâche avec un champ texte et un bouton
  • Afficher la liste des tâches
  • Supprimer une tâche

Contraintes

  • Utiliser HTML, CSS et JavaScript
  • Utiliser des fonctions JavaScript
  • Manipuler le DOM (ex : createElement, appendChild)
  • Organiser le code de manière claire
Conseil : commencez par la logique JavaScript (ajout/affichage/suppression) puis améliorez le design.

Plan de réalisation (recommandé)

1) Modèle de données

Un tableau taches de chaînes (ou d’objets si bonus terminé).

2) Affichage

Une fonction afficherTaches() qui vide la liste puis recrée les éléments.

3) Ajout

Lire le champ, trim(), pousser dans le tableau, puis rafraîchir.

4) Suppression

Un bouton « Supprimer » par tâche (avec l’index), puis rafraîchir.

Structure HTML minimale

<form id="todo-form">
  <input id="todo-input" type="text" />
  <button type="submit">Ajouter</button>
</form>
<ul id="todo-list"></ul>

Exemple de création d’un élément

const li = document.createElement("li");
const p = document.createElement("p");
p.textContent = "Ma tâche";
li.appendChild(p);

Bonus (pour les plus rapides)

  • Marquer une tâche comme terminée
  • Ajouter un bouton « Vider la liste »
  • Sauvegarder les tâches avec localStorage
Astuce : pour la sauvegarde, pensez à JSON.stringify / JSON.parse.

Aide (rappels DOM rapides)

  • Lire un champ : document.getElementById("todo-input").value
  • Vider une liste : liste.textContent = ""
  • Créer : document.createElement("li")
  • Ajouter : parent.appendChild(enfant)
  • Écouter un clic : bouton.addEventListener("click", ...)
Référence : MDN Web Docs.

Récap & suite

  • Un tableau JS = la source de vérité
  • afficherTaches() reconstruit l’UI à partir du tableau
  • Les événements (submit/click) modifient le tableau, puis rafraîchissent
Suite possible : filtres (tout / fait / à faire), recherche, dates, priorité, drag & drop, sauvegarde.
