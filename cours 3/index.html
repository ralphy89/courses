Introduction

Une page web « dynamique » réagit aux clics, à la saisie au clavier et à l’envoi de formulaires grâce au JavaScript dans le navigateur. Vous ciblez des éléments avec document.getElementById("…") (grâce à l’attribut id du HTML), et pour les groupes de boutons radio document.getElementsByName("…") (attribut name), puis vous attachez des écouteurs d’événements (addEventListener).

Ici la connexion et le QCM sont simulés dans le navigateur uniquement : les vrais sites vérifient les identifiants sur un serveur. Ne mettez jamais de vrais mots de passe en clair dans le code public.
Travail en classe (template élève) — Dans le dossier cours 3/templates-eleves/, chaque exercice est une page HTML prête (mise en forme incluse) : l’élève n’a qu’à écrire connexion.js ; pour le QCM, il ajoute les questions dans qcm.html et le script dans qcm.js. Ouvrir l’accueil des fichiers élève.

Rappels — DOM & formulaires

Sélectionner un élément par son id

const form = document.getElementById("login-form");
const message = document.getElementById("message");
const champLogin = document.getElementById("login");

Écouter la soumission d’un formulaire

form.addEventListener("submit", function (event) {
  event.preventDefault(); // évite le rechargement de la page
  const login = document.getElementById("login").value.trim();
  // …
});

Lire le bouton radio coché (même name)

// Les radios d’une question partagent le même name="q1"
function radioCoche(nom) {
  const liste = document.getElementsByName(nom);
  for (let i = 0; i < liste.length; i++) {
    if (liste[i].checked) return liste[i];
  }
  return null;
}

const choisi = radioCoche("q1");
if (choisi) {
  console.log(choisi.value);
}
Utilisez .trim() sur les champs texte pour ignorer espaces en trop. Préférez textContent pour afficher du texte utilisateur (évite l’injection de HTML accidentelle).

Exercice 1 — Page de connexion (simple)

En autonomie : créez connexion.html + connexion.js. En classe avec le template élève, ouvrez templates-eleves/connexion.html : la page est déjà faite, vous complétez seulement connexion.js.

Contenu minimal du formulaire

  • Le <form> doit avoir un id (ex. login-form) pour getElementById.
  • Un champ identifiant (type text, id ex. login).
  • Un champ mot de passe (type password, id ex. password).
  • Un bouton « Se connecter » (type="submit").
  • La zone de message avec un id (ex. message).

Comportement attendu (JavaScript)

  1. À la soumission : preventDefault sur l’événement.
  2. Si identifiant ou mot de passe vide → afficher un message d’erreur dans un paragraphe ou une div (ex. #message), style visible (couleur rouge ou classe CSS).
  3. Sinon, comparer avec des identifiants factices en dur dans le script, par exemple login === "demo" et motDePasse === "code123" (à adapter).
  4. Si correct → message de succès (ex. « Bienvenue ! ») ; sinon → « Identifiants incorrects ».
  5. Option : vider les champs après succès, ou afficher un lien factice « Mot de passe oublié ».
Astuce : gardez le HTML sémantique (<form>, <label for="…"> relié aux id) pour l’accessibilité et le clavier.

Exercice 2 — QCM à réaliser

En autonomie : créez qcm.html + qcm.js (4 questions, 3 choix, radios). Avec le template élève, templates-eleves/qcm.html fournit la structure et le bouton de score : c’est à l’élève d’écrire les questions en HTML, puis d’aligner le tableau bonnesReponses dans qcm.js.

Méthode

Écrivez le HTML des 4 questions à la main (titres, textes, 3 radios par question avec le même name par question : q1, q2… et des value distinctes, ex. a, b, c). Le bouton « Voir mes résultats » et la zone du score ont chacun un id (ex. btn-resultat, resultat).

En JavaScript, un tableau bonnesReponses : une entrée par question, dans l’ordre, avec la value de la bonne réponse pour vos questions (ex. ["a", "c", "b", "a"]). Au clic : parcourez chaque groupe avec getElementsByName("q1"), etc., testez .checked et comparez la value au tableau.

Logique de correction

  1. Compter combien de bonnes réponses (score sur 4).
  2. Afficher le score dans la page (ex. « Vous avez 3 / 4 »).
  3. Afficher un court détail : pour chaque question, « correct » ou « faux » (sans forcément révéler la bonne réponse, ou avec — au choix du formateur).
  4. Si aucune réponse sélectionnée pour une question, gérer le cas (message ou compter comme faux).
Thèmes possibles pour les questions : balises HTML, rôle du CSS, différences let/const, rôle de preventDefault, etc.

Solutions — extraits de référence

À adapter à vos fichiers et à vos identifiants / questions.

Exercice 1 — cœur du script (connexion factice)

const BON_LOGIN = "demo";
const BON_MOT_DE_PASSE = "code123";

const form = document.getElementById("login-form");
const zoneMessage = document.getElementById("message");

form.addEventListener("submit", function (event) {
  event.preventDefault();
  zoneMessage.textContent = "";

  const login = document.getElementById("login").value.trim();
  const mdp = document.getElementById("password").value;

  if (!login || !mdp) {
    zoneMessage.textContent = "Veuillez remplir tous les champs.";
    return;
  }

  if (login === BON_LOGIN && mdp === BON_MOT_DE_PASSE) {
    zoneMessage.textContent = "Connexion réussie — bienvenue !";
  } else {
    zoneMessage.textContent = "Identifiant ou mot de passe incorrect.";
  }
});

Exercice 2 — correction QCM

Adaptez bonnesReponses au contenu réel des questions. HTML : name="q1" … name="q4", value parmi a, b, c. 

const bonnesReponses = ["a", "b", "a", "c"]; // à adapter à VOS bonnes réponses

function radioCoche(nom) {
  const liste = document.getElementsByName(nom);
  for (let i = 0; i < liste.length; i++) {
    if (liste[i].checked) return liste[i];
  }
  return null;
}

function scoreQcm() {
  let score = 0;
  for (let i = 0; i < bonnesReponses.length; i++) {
    const nom = "q" + (i + 1);
    const choix = radioCoche(nom);
    if (choix && choix.value === bonnesReponses[i]) {
      score++;
    }
  }
  return score;
}

document.getElementById("btn-resultat").addEventListener("click", function () {
  const s = scoreQcm();
  document.getElementById("resultat").textContent =
    "Score : " + s + " / " + bonnesReponses.length;
});
