Exercice 2 — QCM

Vous rédigez les questions dans ce fichier HTML ; le calcul du score se fait dans qcm.js (getElementById, getElementsByName).

À faire : qcm.html (au moins 4 questions) + qcm.js (score).

Comment ajouter vos questions

  1. Dans la zone « Questionnaire » ci-contre, sous le titre, insérez au moins 4 blocs <fieldset class="qcm-fieldset">.
  2. Chaque question : un <legend> (l’intitulé), puis 3 <label> contenant chacun un <input type="radio">.
  3. Pour la question 1, tous les radios ont name="q1" ; pour la question 2, name="q2", etc. jusqu’à q4.
  4. Donnez à chaque radio une value parmi a, b ou c (une valeur par proposition).
  5. Ne déplacez pas le bouton #btn-resultat ni la #resultat : le script en a besoin.
  6. Dans qcm.js, remplissez le tableau bonnesReponses : une entrée par question, dans l’ordre q1, q2… avec la value de la bonne réponse (ex. ["a", "c", "b", "a"] selon vos choix).

Exemple d’une question (à copier et modifier)

<fieldset class="qcm-fieldset">
  <legend>1. Votre question ici ?</legend>
  <label><input type="radio" name="q1" value="a" /> Première réponse</label>
  <label><input type="radio" name="q1" value="b" /> Deuxième réponse</label>
  <label><input type="radio" name="q1" value="c" /> Troisième réponse</label>
</fieldset>

Pour la question 2, remplacez q1 par q2 dans name, etc.

Questionnaire

Supprimez ce paragraphe une fois vos questions ajoutées. Ajoutez vos <fieldset> ici (entre ce texte et le bouton).
