Exercice 1 — Connexion

Interface prête : vous codez la logique dans connexion.js (identifiants factices, messages).

Fichier à compléter : connexion.js — utilisez document.getElementById uniquement (pas de querySelector).

Rappel des consignes

  1. À la soumission du formulaire : preventDefault.
  2. Champs vides → message d’erreur dans #message (classe msg-erreur).
  3. Sinon, tester login === "demo" et mot de passe === "code123" (ou changez ces valeurs dans le script, en cohérence avec le cours).
  4. Succès → texte positif et classe msg-succes ; échec → message d’erreur.
