Liste des 10 exercices

Fichiers dans templates-eleves/ — à compléter surtout côté JavaScript.

  1. exo1 — Variables let / const, texte dans la page (textContent).
  2. exo2 — Nombres : lire un champ, convertir avec Number(), calcul simple.
  3. exo3 — Conditions if / else (test sur une valeur saisie).
  4. exo4 — Boucle for : construire une chaîne (ex. nombres de 1 à 10).
  5. exo5 — Tableau : push, parcours, affichage de la liste.
  6. exo6 — Fonction qui return une valeur ; appel depuis un bouton.
  7. exo7 — Objet littéral { }, accès aux propriétés .
  8. exo8 — Chaînes : trim, length, toUpperCase.
  9. exo9 — Clic sur des boutons : addEventListener, modifier le DOM ou une classe.
  10. exo10 — Formulaire : submit, preventDefault, lire plusieurs champs.

Conseils

  • Testez après chaque petite modification (rechargement de la page).
  • Utilisez document.getElementById("…") pour cibler les éléments déjà présents dans le HTML.
  • Préférez textContent pour afficher du texte utilisateur.
  • Gardez une fonction function afficherMessage(msg) { … } si cela vous aide à rester lisible.
